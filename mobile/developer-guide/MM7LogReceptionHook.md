## <a name="MM7LogReceptionHook.purpose"></a> 26.1. Purpose

This hook is used to log MM7 MO messages (e.g. DeliverReq, DeliveryReportReq, ReadReplyReq) received by Mobile Momentum. MM7 MO messages are as follows:

*   DeliverReq

*   DeliveryReportReq

*   ReadReplyReq
