## Name

signing_stats reset — reset signing stats

## Synopsis

`signing_stats reset`

<a name="idp11377936"></a> 
## Description

This command resets all DKIM and Domainkeys signing statistics.

```
10:56:11 ecelerity(/tmp/2025)> signing_stats reset
Signing Statistics are reset
```
