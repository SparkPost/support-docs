# Online Manuals

This section holds the long-form manuals shared with on-premises Momentum
customers — installation guides, upgrade guides, and release notes — for
**Momentum 5.x and later releases**.

Each manual is maintained as Markdown in this documentation repository, so it is:

- **Readable online** here, fully searchable alongside the rest of the Momentum
  documentation; and
- **The single source** from which the branded, downloadable **PDF** edition is
  generated.

## Available manuals

_Manuals are migrated from their source PDFs one at a time. Links appear here
and in the left-hand menu as each manual is converted._
